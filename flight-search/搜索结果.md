## 三、KLM Coupon 专项分析

### Coupon 使用规则（已确认）

| 项目 | 说明 |
|------|------|
| 类型 | KLM Travel Voucher (非退款代金券) |
| 面额 | 约 €120-130 |
| 可用航司 | **KLM、法航 Air France、达美 Delta、维珍大西洋** |
| 可用范围 | 任何航线（**中国/上海不受限**），购票+附加服务 |
| 有效期 | 自签发起12个月 |
| 限制 | 仅限voucher上的姓名使用，不可转让，不可与Flying Blue里程票叠加 |
| 余额处理 | 如voucher未用完，剩余金额会生成新voucher |
| 使用方式 | 在 **klm.com 或 airfrance.com** 结账时输入voucher编号 |

### ⚠ 重要区分

- **Travel Voucher（你的coupon）**→ 无目的地限制 ✅ 可用于上海
- **Discount Code（促销折扣码）**→ 某些code排除中国 ⚠ 注意区分

### Coupon 加持下的价格对比

| 方案 | 原价 | 用coupon(€130)后 | 比其他航司便宜 |
|------|------|-----------------|---------------|
| KLM 9/25-10/8 | €606 | **€476** 🏆 | 比TK便宜€170+ |
| KLM 9月中旬 | €650-750 | **€520-620** | 比LH便宜€100+ |
| KLM 7-8月旺季 | €800-1100 | **€670-970** | 与TK/LH持平或更低 |
| AF 法航(同价) | 同KLM | **同KLM** | coupon也可用于AF |

> 💡 **结论：有了coupon，KLM/AF成为全时段碾压性的最优选择。即使旺季，用coupon后的KLM也比淡季的TK/LH便宜。**

## 五、行动指南

### 🎯 第一优先：KLM + Coupon（推荐立即操作）
1. 去 **klm.com** (或 klm.de)，搜 NUE→PVG，试以下日期组合：
   - **9/25 → 10/8**（2周，底价€606，coupon后€476）
   - **9/15 → 10/6**（3周）
   - **9/1 → 9/29**（4周）
   - **9/1 → 9/22**（3周）
2. 在支付页面输入 **coupon/voucher编号** 抵扣€120-130
3. 注意选择有AMS→PVG **直飞**的航班（避免AMS→CDG→PVG两次中转）

### 📋 对比验证
4. 去 **Google Flights** 搜同样日期，确认KLM价格是否最低
5. 也在 **airfrance.com** 搜一下——同集团可能有不同组合，coupon也能用
6. 如KLM价格偏高，再看 **turkishairlines.com**（10月TK最便宜，但不能用coupon）

### 购买时机
- **现在就可以买**（提前5-6个月，价格合理区间）
- 或设置价格追踪等1-2周看是否有更低价
- ⚠ 注意coupon有效期——确认你的coupon在购票时仍在12个月有效期内
- ⚠ 不要等太久——暑假前价格只会越来越高

### ⚠ Coupon 使用注意事项
- **必须在 klm.com 或 airfrance.com 官网购票**才能用coupon
- 不能在 Google Flights / Skyscanner / Trip.com 等第三方平台使用
- coupon不能和Flying Blue里程票叠加
- 如果票价低于coupon面额，剩余金额会生成新voucher
- 购票后如退票，coupon抵扣部分不退现金（仅退差额部分）
